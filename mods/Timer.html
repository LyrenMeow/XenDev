Timer

It is essential to have loading times removed from the mod you are playing:

1. Right-click LiveSplit and press Edit Splits

2. Change the game name to Half-Life 2 Mods (Some mods work with their own respective name)

3. SourceSplit options should appear above the splits. Press Activate

4. Press Settings. In the Settings, see that the Game Time is set to Engine Ticks with Pauses

5. Press Ok in the SourceSplit window and Splits Editor

6. Right-click LiveSplit, hover over Compare Against and select Game Time. If the Game Time doesn’t work, it’s because of your Layout Settings. In your layout, you have to compare against either “Current timing method” or Game time.

7. Save your layout, save your splits. This way you don’t have to setup everything for the second time, only some of the steps when you are going to run another mod. It is recommended toduplicate splits from another HL2 mod to a new one so it’s easy to skip steps 2-5.

8. When you relaunch your LiveSplit, do it from the LiveSplit’s executable, not through the splits/layout file. This way it saves all the settings, such as which splits you had last time, which layout you used and it saves the setting for step 6.

Some mods have an automatic timer that works only with 2838's version of the SourceSplit, you can get it from here:

https://github.com/thisis2838/SourceSplit/releases

Replace your LiveSplit.SourceSplit.dll with the one from the link above in your LiveSplit's /Components folder.
